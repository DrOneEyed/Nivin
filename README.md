# Nivin — Android P2P Chat Application

A peer-to-peer real-time chat application for Android, enabling direct messaging between users without a centralised server.

## Features

- Real-time peer-to-peer messaging
- User registration and profile setup
- Message history with timestamps
- Clean, native Android UI
- Lightweight with minimal permissions

## Getting Started

### Prerequisites
- Android Studio (Electric Eel or newer)
- Android SDK 21+

### Build & Run

```bash
git clone https://github.com/DrOneEyed/Nivin.git
```

1. Open the project in **Android Studio**
2. Let Gradle sync complete
3. Connect a device or start an emulator
4. Click **Run**
